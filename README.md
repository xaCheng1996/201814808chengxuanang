### 任务1：KNN

核心程序文件有两个:

KNN.py

Vector_space_model.py

KNN包含相似度计算、分类和acc计算

Vector_space_model包含向量空间构建。

辅助文件一个：

Inverted_index.py

Inverted_index.py主要用于将原始数据转化为倒排索引格式，原始数据见开发日志分享的谷歌云链接。

运行方式：

`python3 KNN.py`

`（保证data_tf_idf_inverted文件夹与KNN.py在同目录）`
